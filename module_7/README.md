# Проект "Выбираем авто выгодно"

## Цели и задачи проекта
### Цель
Цель проекта - создать модель, которая по характеристикам определит рыночную стоимость автомобиля с пробегом.

### Задачи
1. Собрать данные об автомобилях с пробегом, т.к. данные для обучения отсутствуют. Сбор данных будет проведён с порталов автообъявлений.
2. Проанализировать и очистить данные, првести разведывтельный анализ собранных данных.
3. Создать несколько моделей из разных регрессоров и провести моделирование с выявлением наилучшей модели.
4. С помощью лучшей модели предсказать цены на автомобили из тестовой выборки.

### Средства выполнения задач
Sweetviz библиотека для разведывтельного анализа, python библиотеки для машинного обучения. Использованные регрессоры:
1. LinearRegression
2. RandomForestRegressor
3. ExtraTreesRegressor
4. StackingRegressor
5. GradientBoostingRegressor
6. BaggingRegressor с базовым регрессором LinearRegression
7. AdaBoostRegressor с базовым регрессором LinearRegression

### Файлы проекта

1. Final work.ipynb - ноутбук с выполнеными задачами.
2. requirements.txt - требования к python окуржению.
3. в каталоге data сложены данные для обучения и тестовые данные для генерации итогового датасета (взято у команды Car Price prediction_team_LAE. Превеликая им благодарность :)):
   1. 09_09_2020.csv.zip - основной датасет для обучения
   2. train_df_full.csv.zip - дополнительно собранные данные для обучения
   3. test.csv.zip - тестовая выборка для генерации конечного результата.

### Запуск
Рекомендуется скачать все файлы проекта на локальный компьютер, установить пакет программ [Anaconda](https://www.anaconda.com), распаковать файлы с данными рядом с ноутбуком и запустить ноутбук Final work.ipynb.
